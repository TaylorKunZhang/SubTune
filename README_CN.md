SubTune 是一个免费的开源音乐流媒体 Android 客户端，适用于 Subsonic API（1.13.0 或更高版本）兼容服务器（[Subsonic](http://www.subsonic.org/pages/index.jsp)、[Navidrome]( https://www.navidrome.org/)、[Airsonic](https://airsonic.github.io/) 等）。

- Material 3 design
- 浏览专辑和歌单
- 搜索专辑和歌曲
- 设置最大比特率
- 深色模式
- 动态颜色
